
# QC Load Dashboard

This is a ready-to-use GitHub Pages dashboard for QC Load analytics.

## Features
- Month-wise QC analysis
- Week-wise QC analysis
- Day-wise QC analysis
- Enterprise-wise QC load
- Team-wise QC load
- Filters for Enterprise and Team
- KPI summary cards

## Deployment Steps

1. Upload all files to a GitHub repository
2. Go to:
   Settings → Pages
3. Select:
   Deploy from Branch → main → root
4. Save

Your dashboard will be live in 1-2 minutes.
